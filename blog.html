What is the Document Object Model(DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects, where each part of the document (like elements, attributes, and text) is accessible and manipulatable using languages like JavaScript. Through the DOM, developers can dynamically update content, structure, and styles on a webpage.

How do select an element from the DOM?

To select an element from the DOM, you can use methods like:

1.document.getElementById('id') Selects an element by its ID.
2.document.querySelector('selector') Selects the first element that matches a CSS selector.
3.document.getElementsByClassName('class') Selects all elements with a specific class.
4.document.getElementsByTagName('tag') Selects all elements with a specific tag.

What is event delegation in the context of the DOM, and what is it useful?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element to handle events from its child elements. Instead of adding separate listeners to each child, the parent captures events that bubble up from its children, then determines which child triggered the event. This is useful for:
Efficiency: Fewer event listeners mean better performance, especially with many child elements.
Dynamic content: It works even when child elements are added or removed dynamically

How do You manipulate an element's attributes and styles using the DOM?

o manipulate an element's attributes and styles using the DOM:
1. Attributes:
Get an attribute: element.getAttribute('attribute')
Set an attribute: element.setAttribute('attribute', 'value')
Remove an attribute: element.removeAttribute('attribute')
2. Styles:
Inline styles: Use element.style.property = 'value' (e.g., element.style.color = 'red')
CSS classes: Use element.classList.add(), element.classList.remove(),
or element.classList.toggle() to manipulate CSS classes.
